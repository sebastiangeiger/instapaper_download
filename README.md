# instapaper_download

This gem provides a binary that lets you download an epub from instapaper and put it on an ebook reader.

## Installation

    $ gem install instapaper_download

## Usage

If neccessary do 

    $ rvm reload

then

    $ instapaper_download

No parameters or anything!
